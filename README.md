##  Features Completed
 **Responsive Navbar:** Includes a left-aligned logo, right-aligned links, and a        mobile-friendly hamburger menu.

 **Hero Section:** Features a high-conversion headline and a functional CTA button.

 **Services Grid:** Displays 3 distinct service cards using CSS Grid/Flexbox that stack vertically on mobile.

 **Dark/Light Theme:** Implemented using vanilla JavaScript and CSS variables.

 **Hover Effects:** CTA buttons scale up and change color; Service Cards feature a 3D Z-axis lift effect on hover.

 **Sticky Navbar:** The navigation bar remains fixed at the top of the viewport during scrolling.
